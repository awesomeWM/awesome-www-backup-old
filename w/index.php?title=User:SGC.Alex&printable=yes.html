awesome and I

awesome discovery

I’ve heard of awesome the first time circa june 2009 from an awesome user who was thrilled by this wm. At that time, I was using Gnome, and I’ve been astonished by the impression of awesome clarity, so I did install it on my system, for testing purposes. Soon, I figured out a way of getting my GTK theme back, added nm-applet, gnome-power-manager and a keyboard layout manager — bépo (French version of Dvorak) is awesome, too — on my statusbar, then decided to keep awesome as my main window manager.

awesome usage

I run awesome on Debian sid and I use pcmanfm as file manager.
