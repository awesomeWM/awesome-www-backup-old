Vous trouverez ci-dessous une liste des articles décrivant les instructions pour installer awesome sur différentes plate-formes. Dans le pire des cas, vous pouvez toujours compiler la version git d’awesome sur n’importe quelle plate-forme *NIX via
git clone git://git.naquadah.org/awesome.git ou git clone http://git.naquadah.org/git/awesome.git.

Nous vous conseillons ensuite de lire quelques informations pour bien débuter avec awesome.


Arch Linux
  • awesome 3 sur Arch Linux : installer awesome sur Arch Linux


Aurora UX
  • awesome 3.1.1 sur AuroraUX : compiler awesome sur AuroraUX


Debian
  • awesome 3 sur Debian Lenny : installer awesome 3 sur Debian 5.0 Lenny (« stable » actuelle)
  • awesome 3.3.1 sur Debian testing (« Squeeze ») ou sid : installez simplement le paquet awesome
  • awesome git sur Debian : compiler des paquets Debian depuis la version git d’awesome


FreeBSD
  • awesome 3.3.1 sur FreeBSD : port pour installer awesome sur FreeBSD


Gentoo
  • awesome 3.3.1 sur Gentoo : installer les versions 3 ou git d’awesome avec gestionnaire de paquets
  • awesome git sur Gentoo : compiler localement la version git d’awesome


Mandriva
  • awesome 3.3 sur Mandriva Cooker : installer awesome 3.3 sur Mandriva Cooker


OpenSuse
  • awesome 3.3.1 sur OpenSuse : paquets .rpm et sources pour installer ou compiler awesome 3.3.1 sur OpenSuse


Slackware
  • awesome 3 sur Slackware 12.2 : SlackBuilds pour installer awesome sur Slackware


Source Mage GNU–Linux
  • awesome 3.2.1 sur Source Mage GNU–Linux : archive à installer pour Source Mage GNU–Linux


T2SDE
  • awesome 3.2.1 sur T2SDE : archive à installer pour T2SDE


Ubuntu
  • awesome 3.3.1 sur Ubuntu 8.04 : compiler awesome sur Ubuntu 8.04 Hardy Heron
  • awesome 3.3.1 sur Ubuntu 9.10 : installez simplement le paquet awesome sur Ubuntu 9.10 Karmic Koala
  • awesome git sur Ubuntu 8.10 : compiler la version git d’awesome sur Ubuntu 8.10 Intrepid Ibex
