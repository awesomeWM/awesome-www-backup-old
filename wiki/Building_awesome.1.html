FreeBSD[edit]

 cd /usr/ports/x11-wm/awesome && make install clean

(/usr/ports/x11-wm/awesome2 is still available)

Sabayon Linux[edit]

(As root, in a terminal)

  • Entropy version:
 equo update && equo install awesome
  • Portage version:
 emerge [options] awesome

PLD Linux[edit]

Run install awesome in poldek (or run poldek -i awesome).
