setxkbdmap command[edit]

You can divide its duties into two sections:

  • Get result
  • Set key map

Get result[edit]

setxkbdmap command can give you useful information about keyboard. Practical: 

 root@debian:/home/mohsen# setxkbmap -query
 rules:      evdev
 model:      inspiron
 layout:     us

suppose you want to retrive current layout from setxkbdmap command, You can do it: 

setxkbmap -query |grep layout |awk -F: {'lay=$2;sub(/^[ \t\r\n]+/, "", lay);sub(/[ \t\r\n]+$/, "", lay);print lay'}

Along with -query option, You can use the <codde>-print</code> option for getting X information: 

root@debian:/home/mohsen# setxkbmap -print
xkb_keymap {
	xkb_keycodes  { include "evdev+aliases(qwerty)"	};
	xkb_types     { include "complete"	};
	xkb_compat    { include "complete"	};
	xkb_symbols   { include "pc+us+inet(evdev)"	};
	xkb_geometry  { include "pc(pc104)"	};
};

Set key map[edit]

You can set you keyboard layout with setxkbdmap command , For example: 

setxkbdmap us ##Switch to English language
setxkbdmap ir ##Switch to Persian language
setxkbmap ru  ## Switch to Russian language
setxkbmap jp ##Switch to Japanese language

However you can set many parameters with setxkbdmap foor your language, if you want, please hack man 1 setxkbdmap

xmodmap command[edit]

Sure, You're familar with modifier keys:
⇧ Shift Shift key
Ctrl(Control)
Alt(Alternate) – also labelled ⌥ Option on Apple Macintosh keyboards
AltGr(Alternate Graphic)
♦– Meta key, found on MIT, Symbolics, and Sun Microsystems keyboards
Windows Log found on Windows keyboards (However if you read other documentation, you prevent other name such as super key in Gnome documentations.)
⌘ command– Command key, found on Apple Macintosh keyboards. On older keyboards marked with the Apple logo.
Fn(Function) – present on small-layout keyboard, usually on notebooks.
