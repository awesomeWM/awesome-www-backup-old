FS#1260 - Delegation to normal event handler from keygrabber

Attached to Project: awesome

Details

It's great if the last function of keygrabber can delegate event handling to the normal event handler, i.e. global or client key handler.

Without such a feature, I have to write helper functions to enable/disable event handling when I want a handler which works only if some key is being pressed:
awful.key({'Any'}, 'Mod4', function_to_enable_handler, function_to_disable_handler)
awful.key({'Mod4'}, 'x', function() if handler_enabled then do_handle_event() end)
