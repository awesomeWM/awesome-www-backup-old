Details

For some reason, awful.tag.getidx() returns nil in rev 779d43fc462411d84adf757a178e938c6ba9d3b8, while in 8348d444446d1fa0ebfc8392ee01ae058b8c1966 it still worked fine.

I'm using the function in a awful.key callback and a (single) tag is selected. I'm not sure if this is important in some way, but I'm only using one screen.
awful.tag.selected() seems to be working, at least it is not returning nil. Manually going through a loop and comparing the tags with awful.tag.selected() works fine.

You should be able to replicate this by defining a new key-binding which creates a notification-box or something:
awful.key({ modkey }, "a", function() naughty.notify({ preset = naughty.config.presets.critical, title = "awful.tag.getidx()", text = tostring(awful.tag.getidx()) }) end)
