Comment by Pawe? Zuzelski (pawelz) - Saturday, 04 July 2009, 16:27 GMT
I've encountered the same issue. It was somehow related to awful.hooks.timer.register.

Workaround was to comment out the following code in my rc.lua:

-- Hook called every second
awful.hooks.timer.register(1, function ()
-- For unix time_t lovers
-- datetime.text = " " .. os.time() .. " "
-- Otherwise use:
datetime.text = " " .. os.date() .. " "
end)

-- Hook called every 5 seconds
awful.hooks.timer.register(5, function ()
battery.text = " " ..
execute_command("~/.check-battery.sh") .. " "
end)
