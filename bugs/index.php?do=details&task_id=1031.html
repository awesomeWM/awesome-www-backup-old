Details

I'm using Awesome on Fedora 16. After install Awesome through Yum, which worked fine, Awesome is now showing up in Gnome's application menu. I filed a bug with Gnome's bug tracker[1] about this and the response was that Awesome's .desktop file should include a line to mark it as "Hidden" so it won't show up in Application menus.

I looked through Awesome's git repository and found the .desktop file [2]. It does not include this line. The expected resolution to be include this line to prevent this issue.

[1] https://bugzilla.gnome.org/show_bug.cgi?id=682132
[2] http://git.naquadah.org/?p=awesome.git;a=blob;f=awesome.desktop;h=8a7f354cbd286c5acc31ce517cf3e9efb4676a38;hb=HEAD
