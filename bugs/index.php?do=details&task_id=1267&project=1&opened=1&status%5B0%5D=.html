Details

See https://github.com/copycat-killer/lain/issues/48

Because awful.mouse.resize and awful.mouse.client_resize_tiled check a hardcoded list of layouts, it's impossible for any thirdparty layout to easily support mouse resizing.

Possible avenues:
1) allow a custom layout to specify a callback to be used on mouse.resize
2) allow a custom layout to specify that it should be treated the same way as some stock layout. This is less work for awesome, less work for custom layout devs, and probably works in a majority of cases.
