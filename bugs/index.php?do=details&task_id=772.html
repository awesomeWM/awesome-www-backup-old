After upgrade to 3.4.5 I expirience (at least) two visual artifacts:

- really awful flickering when redrawing or even switching windows (I see it with rxvt-unicode especially)

- strange redrawing wibox artifa?ts (see attached screenshot): there're "shadows" of previously drawn "taskbar" items, disappearing after several seconds or any window action.

This was reproduced on two Gentoo Linux ~amd64 boxes, both running the latest packages from Portage tree (Xorg server 1.8, for example, and 2.6.33 kernel) with kernel modesetting enabled (radeon and nouveau). Downgrade to 3.4.4 solves both issues.
