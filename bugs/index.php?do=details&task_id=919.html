Comment by Uli Schlachter (psychon) - Tuesday, 23 August 2011, 07:59 GMT
Sorry, totally overlooked that. Anyway, this window doesn't have any _NET_WM_STRUT or _NET_WM_STRUT_PARTIAL set. Seems like only some user xfce4-panel has reserved 32 pixel at the left screen edge (the one that your previous command showed).
So xfce4-panel isn't asking for not being covered by windows. If you can find the client for xfce4-panel, you could set your own struts with c:struts({ left = 5, right = 5, top = 5, bottom = 5}). (Numbers are made up, I'm not sure what the right numbers would be. Perhaps c:struts({ right = c.geometry().width })?)
