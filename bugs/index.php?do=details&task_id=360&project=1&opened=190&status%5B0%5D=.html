updated to v3.0-241-g827d084 from a version a couple of days old.

Now wibox background color doesn't seem to get set until there's a widget inserted into to it. To illustrate - I have a 3 wiboxes on the top of the screen: widgetbar, separatorbar, and tabbar. The middle one is just an empty, 3 pix high strip serving as a spacer between the other two. Here's the code:

widgetbar = {}
for s = 1, screen.count() do
widgetbar[s] = wibox({ position = "top", name = "widgetbar" .. s,
fg = beautiful.fg_normal, bg = beautiful.bg_normal, height = 16 })
-- Add widgets to the statusbar - order matters
widgetbar[s].widgets = jointables(widgets_left,widgets_right)
widgetbar[s].screen = s
widgetbar[s].ontop = true
end

separatorbar = {}
for s = 1, screen.count() do
separatorbar[s] = wibox({ position = "top", name = "separatorbar" .. s,
fg = beautiful.fg_normal, bg = beautiful.bg_normal, height = 3 })
separatorbar[s].screen = s
separatorbar[s].ontop = true
end
