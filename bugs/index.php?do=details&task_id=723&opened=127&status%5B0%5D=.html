Comment by Andrei Thorp (garoth) - Wednesday, 17 February 2010, 17:47 GMT
This feature is probably implemented using this standard: http://standards.freedesktop.org/wm-spec/wm-spec-1.3.html#id2506667

However, implementing this feature is optional. Also, there isn't really much point in showing the desktop in Awesome... I mean, there is no "desktop" really. It's just empty space. I don't know exactly why people would want this.

As a final point, it is fairly easy to implement this feature in an awesome-specific way. In fact, I have already written functions that will minimize all the windows currently being displayed, or unminimize them all. I can dig out the code if someone wants.
