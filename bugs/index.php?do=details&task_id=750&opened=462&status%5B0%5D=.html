Details

Pango markup in menu entries is not processed, e.g. if I invoke

echo 'awful.menu.new { items = { { "<b>test</b>" } } }:show()' | awesome-client

in a terminal, the menu shown contains the text "<b>test</b>". Pango markup in other cases, e.g. naughty or widgets, works fine, and until 3.4.3 also worked fine with menus.
