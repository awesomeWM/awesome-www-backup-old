Comment by Uli Schlachter (psychon) - Sunday, 30 March 2014, 21:47 GMT
The new change is that clients become urgent if they try to steal focus and that fails because they don't have the right tag where previously such a request was silently ignored.

Do you really want the new client to be "hidden somewhere" without you knowing (for sure) that this happened?
The easiest way that I can think of right now to get that behavior back would be to change the default rule from awful.rules in your config. Instead of
properties = { focus = awful.client.focus.filter }
you would do
callback = function(c) if awful.client.focus.filter(c) then client.focus = c end
