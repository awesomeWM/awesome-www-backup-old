Details

Java application stay grey until I try to resize it, then I can click randomly around to reveal more of the window. This bug only occurs with SUMlauncher (An utility to update the game Overgrowth).

It works fine in 3.4.13 but it doesn't work with git/master, so I guess it's a bug in awesome. I tried wmname LG3D and JAVA_AWT_WM_NONREPARENTING=1 but none of those helped.
