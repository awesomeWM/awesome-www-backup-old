Details

_NET_WM_WINDOW_TYPE for awesome docks should be _NET_WM_WINDOW_TYPE_DOCK.
_NET_WM_WINDOW_TYPE for awesome menus should be _NET_WM_WINDOW_TYPE_POPUP_MENU.
_NET_WM_WINDOW_TYPE for naughty notifications docks should be _NET_WM_WINDOW_TYPE_NOTIFICATION.

This is requirement of EWMH: http://standards.freedesktop.org/wm-spec/wm-spec-1.4.html#id2551529
And this will help compositing manager to correctly recognize window type and add proper decoration.

Step to reproduce:
$ awesome & xcompmgr -cC
You will be able to see shadow under dock (if you have it on the top of screen).

P.S. Titlebar should have _NET_WM_WINDOW_TYPE too... But I don't know what value is more suitable for this.
