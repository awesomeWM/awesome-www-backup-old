Details

Currently, some information like visual, SN monitor, client_focus or systray are stored in screen_t struct.

But this data belongs to physical screen, this means that if you have 3 screen_t with 1 physical screen (Xinerama mode), we only use screens.tab[0] data.

We should add a physical_screen_t (with an array of it) inside globalconf to store this data.
