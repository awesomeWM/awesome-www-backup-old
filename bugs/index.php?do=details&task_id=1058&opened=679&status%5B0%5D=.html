Details

Right now we have a category named "Awesome 3" on the wiki that unites all version3-related pages. After 3.5 is released we no longer can use this category since 3.4 and 3.5 are largely incompatible.

The task is to create separate 3.4 category (since many users will still use it for quite long), review most important wiki pages and separate those that are incompatible from those that aren't.

The issue is created just to be sure it is done before the final release.
