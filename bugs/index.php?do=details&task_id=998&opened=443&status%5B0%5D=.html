Details

When pressing Mod4+R in the default config, a Run prompt is shown. You can use the up arrow to view previous commands, however it is very unpredictable which commands get added to this list. Usually invalid commands are added, but most valid commands are not.

This means you can run a bunch of commands but pressing Mod4+R then Up won't show any of them.

I'm wondering whether there's a list of commands somewhere, because "emacs" (valid command) never gets added to the history, but "xterm" (invalid command - I don't have it installed) never gets added to the history either. But both "yterm" and "xterm2" get added to the history, and both are invalid commands.
