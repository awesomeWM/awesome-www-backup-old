Details

When a wibox is vertical, all widgets in it are rotated 90Â°. This can be kind of ugly and wasteful of space. Especially if you make the wibox very wide (~120px), the task list becomes a giant pile of icons, and other widgets expand to needlessly large sizes. Changing layouts doesn't seem to have any effect.

With Xfce4-panel and Windows taskbar for example, you can make the panel vertical and widgets remain horizontal, to make nice layouts like the one pictured here. It'd be great to do similar with Awesome widgets. Maybe it's possible to have a stack of horizontal wiboxes inside a vertical one?
