Details

If you focus away from a wine window it looses keyboard input when refocused. For keyboard to work again, I need to click the client.

To reproduce:
1. run a wine app (eg. notepad)
2. the client is managed, focused and accepts input
2. focus away
3. focus back without clicking the client (sloppy, tag change)
4. the client is focused but accepts no input until clicked

Tested with Notepad, Foobar2000 and Ableton Live.
Experienced throughout awesome 3.0rc6 - 3.0+git20081006 with both custom and default configs.
Wine version 1.1.4 and 1.1.5
