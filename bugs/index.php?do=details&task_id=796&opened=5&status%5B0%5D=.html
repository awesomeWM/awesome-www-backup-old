Comment by Ignas Anikevicius (gns_ank) - Tuesday, 14 September 2010, 10:46 GMT
Now I get what this bug is about.
Prompt can not handle characters produced with a keystroke sequence (i.e. to write á at first a key producing accent is pressed and later letter a is pressed).
Prompt does not have support for so called deadkeys.

Related wiki article:
http://en.wikipedia.org/wiki/Dead_keys

I guess, that the promptbox tries to interpret any key press it detects and deadkeys are not being interpreted as they should.
