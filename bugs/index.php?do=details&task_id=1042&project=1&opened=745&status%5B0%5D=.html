Hi,

I have just started using awesome and seem to have run into an odd bug.

When I switch between tags, randomly my gnome-terminal will not receive focus properly.
In the attached screenshot, the cursor should be black and flashing, instead it is an unfilled rectangle. Strangely, I can still type in the terminal and it receives the keystrokes.

It seems to happens most but not all of the time and the only way to fix it is to move back and forth between tags and switch focus between various windows until the terminal gets the focus correctly.

The same thing happens with my Empathy chat windows. I switch tags to type into it and can type in the box, but I can't see any cursor which is rather frustrating.

I'm running Arch Linux. I tried the packaged awesome 3.4.13 and also git/master (v3.4-770-g77243cd) with the stock configurations as well as with disabling sloppy focus. However, all give the same result.

#971 seems sort of related, but changing between windows doesn't fix the problem, nor does running git/master.

Thanks for any solution to the problem!
